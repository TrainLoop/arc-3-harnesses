## Game dc22 Overview

This is a **puzzle-platformer** where you move a **pink 2x2 player** (`rpygrnbjhwj1`, color 14) to reach a **yellow 2x2 goal** (`bqxa`, color 11). The win condition (`uhqbwfdkff` at line 2933) is simply: player position == goal position.

**Controls:**
- Actions 1-4: Move UP/DOWN/LEFT/RIGHT (2 pixels per step, `qqsswktiui=2`)
- Action 6: **Click** on UI elements (buttons, crane controls)

**Core Mechanics:**
- **Platforms** (`kbqq-efzv`): Red (color 2) floor tiles the player walks on
- **wbze (bridges)**: Multi-variant terrain pieces. Each has a numbered suffix (1, 2). Only one variant is active at a time; the rest are REMOVED
- **jpug (buttons)**: Colored clickable buttons on the right side. Each has a single-letter tag (a/b/c/d) that controls which group of sprites it toggles
- **Falling**: If the player steps off a platform (no `itki` sprite underneath), they lose **20 steps** and reset to checkpoint
- **Step counter**: Each level has a limited budget. Each move costs 1 step
