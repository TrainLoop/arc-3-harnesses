## Game Mechanics Summary

This is a **connector puzzle**. Each sprite has special **color-8 pixels** that act as connectors. The goal is to **position sprites so all color-8 pixels overlap pairwise** (exactly 2 sprites share each 8-pixel world position). When all 8-pixels are connected, the level is won.

- **Actions 1-4**: Move selected sprite (UP/DOWN/LEFT/RIGHT)
- **Action 5**: Rotate selected sprite 90° clockwise
- **Action 6**: Click to select a different sprite (requires display x,y coordinates)

The initially selected sprite is the one closest to the origin (min x²+y²).

**Important**: ACTION6 requires click coordinates. In the action arrays below, `6` is followed by two integers representing display pixel coordinates (x, y) for the click target. The harness needs to consume these as `ActionInput(id=ACTION6, data={"x": dx, "y": dy})`.
