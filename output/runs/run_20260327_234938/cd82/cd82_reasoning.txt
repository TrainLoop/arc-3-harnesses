## Game Mechanics Analysis

This is a paint-mixing puzzle. The game has:

**Layout:**
- A 10x10 **canvas** (`xytrjjbyib`) starting all black (color 0), placed at grid position (27, 34)
- A **basket** that moves around 8 positions in a ring (like compass points around the canvas)
- **Color buttons** (`pqkenviek`) for selecting paint colors
- A **gate** (`ctwspzkygu`) on levels 3-6 for painting smaller rectangles
- A **target pattern** (`eoqnvkspoa-pqwmeN`) to match

**Basket Positions** (mapped to a 3x3 grid, center excluded):
```
pos7(0,0)  pos0(0,1)  pos1(0,2)
pos6(1,0)    [X]      pos2(1,2)
pos5(2,0)  pos4(2,1)  pos3(2,2)
```

**Actions:**
- 1-4 (UP/DOWN/LEFT/RIGHT): Move basket on the 3x3 grid (center blocked)
- 5 (ACTION5): Pour paint from current basket position onto canvas
- 6 (ACTION6): Click â€” selects colors or triggers gate (requires coordinates)

**Pour regions** (Action 5 from each position):
| Position | Type | Region painted |
|----------|------|----------------|
| 0 (top) | horizontal | rows 0:5, all cols |
| 1 (top-right) | diagonal | row i: cols i:10 (upper-right triangle) |
| 2 (right) | horizontal | all rows, cols 5:10 |
| 3 (bottom-right) | diagonal | row i: cols (9-i):10 (lower-right triangle) |
| 4 (bottom) | horizontal | rows 5:10, all cols |
| 5 (bottom-left) | diagonal | row i: cols 0:(i+1) (lower-left triangle) |
| 6 (left) | horizontal | all rows, cols 0:5 |
| 7 (top-left) | diagonal | row i: cols 0:(10-i) (upper-left triangle) |

**Gate pours** (Action 6 click on gate, levels 3-6):
| Gate Position | Region painted |
|---------------|----------------|
| 0 | rows 0:3, cols 3:7 |
| 2 | rows 3:7, cols 7:10 |
| 4 | rows 7:10, cols 3:7 |
| 6 | rows 3:7, cols 0:3 |

**Win condition:** Canvas must match target at all positions EXCEPT the two diagonals (main diagonal `(i,i)` and anti-diagonal `(i, 9-i)`). That's 80 of the 100 pixels that must match.

**Important limitation:** Action 6 requires click coordinate data. The current harness sends `ActionInput(id=ga)` without data, so Action 6 does nothing. Only Level 1 (which needs just colors 0 and 15) is fully executable. For levels 2-6, I include Action 6 as a placeholder for color/gate selection; the harness would need extension to pass coordinates.
